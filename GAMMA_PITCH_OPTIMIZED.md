# Edulink: Revolutionizing Student Discipline Management

---

# The Challenge

Traditional discipline tracking is inefficient

- Manual record-keeping is time-consuming and error-prone
- Teachers struggle to identify students quickly
- Lack of real-time data for decision-making
- No centralized system for tracking student behavior

---

# Introducing Edulink

AI-Powered Student Discipline Tracking System

A production-ready web application that revolutionizes how schools manage student discipline using cutting-edge face recognition technology.

**Edulink** empowers teachers to identify students instantly and track Sahsiah (discipline) points seamlessly.

---

# Key Features

## Instant Face Recognition
- Identify students in seconds using InsightFace buffalo_l model
- State-of-the-art accuracy with 512-dimensional embeddings
- Works seamlessly with mobile phone cameras

## Comprehensive Discipline Tracking
- Record rewards and punishments with automatic point calculation
- Real-time dashboard with student statistics
- Complete history tracking for each student

## Secure & Scalable
- JWT-based authentication system
- Role-based access control for teachers
- Production-ready architecture

## Mobile-First Design
- Responsive UI optimized for phone cameras
- Works on any device, anywhere
- Intuitive interface for teachers

---

# Technology Stack

## Backend
- **FastAPI** - High-performance Python framework
- **MySQL** - Robust database with SQLAlchemy ORM
- **InsightFace** - State-of-the-art face recognition
- **JWT Authentication** - Secure token-based auth

## Frontend
- **React 18** - Modern UI framework
- **Tailwind CSS** - Beautiful, responsive design
- **React Query** - Efficient data management
- **Mobile-Optimized** - Perfect for on-the-go use
